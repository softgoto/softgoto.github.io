Nexus介绍

Nexus 是个仓库管理器，目前主要分 2 大版本：2.X 和 3.X，其中 2.X 主要支持的格式是 Maven、P2、OBR、Yum。3.X 主要支持的是 Docker、NuGet、npm、Bower、Pypi、Ruby Gems，当然也支持构建工具 Maven 和 Gradle。Nexus 3 只支持 Oracle jdk8，不支持其它版本的 JDK，比如 OpenJDK。
